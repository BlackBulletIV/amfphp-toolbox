Packagecom.novafusion.net.amfphp
Classpublic class Call
InheritanceCall flash.events.EventDispatcher

Since : 2.0 beta

This object represents a unique call to AMFPHP. It holds it's service string and an array of parameters to send to AMFPHP. This object also has a responder of it's own, and holds it's own results and faults. This allows for no overlapping confusion with multiple calls.

addListeners()method
public function addListeners(resultListener:Function = null, faultListener:Function = null):Call

Adds listeners for the result and fault events when this call is called. You can add multiple listeners if you want. If you leave one of the parameters as null, it won't be added.

Parameters

resultListener:Function (default = null) — A function that you want to be called when a result is received.
 
faultListener:Function (default = null) — A funciton that you want to be called when a fault is received.

removeListeners()method 
public function removeListeners(resultListener:Function = null, faultListener:Function = null):Call

Does the opposite of addListeners(). If one of the parameters is null, the removal of the listener won't go ahead.

faultListener:Function (default = null) — A function that you added before as a listener for faults and now want to remove.
