Classpublic class CallEvent
InheritanceCallEvent flash.events.Event

Since : 2.0 beta

A custom event class for calls (results or faults). This is needed so that extra information about the call that triggered it is available. Like the callId and the result object. This class also stores the constants for the triggering of the custom events.
