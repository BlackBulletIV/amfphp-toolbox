Classpublic class CallQueue
InheritanceCallQueue Object

Since : 2.0

A class that acts a queue for calls. The queues form in a line, and each time a call receives a result or fault, the queue moves up one. You can change the number of calls that can be in the "hanging" state (meaning they are waiting for a result or fault) at one time by changing the concurrentCalls property.

concurrentCallsproperty
public var concurrentCalls:uint = 1

The number of calls that can be called on each check. In other words, how many calls can fit through the tube at once?

The default value is 1.

keepDoneCallsproperty 
public var keepDoneCalls:Boolean

Whether or not to keep the calls that have already passed through the queue.

call()method 
public function call():CallQueue

The main method of the object. This method will check to see if it can move anymore call objects out of the waiting line and call them. Generally you won't have to deal with this method, as it is called by whenever a call receives a result/fault (meaning the queue has moved up one). But if you want to force a re-check, then you can call it.
