Classpublic class CallSet

A class to help with making a multiple calls at once. You can add calls to the class, and call them all at once, or setup a timer to call them all on a recurring basis.

The constructer of the set, which sets up the needed properties for the object to operate. You can specify the call objects right here after the connection object; they need to be of type Call.

public function setupTimer(waitDuration:uint = 1000, startNow:Boolean = false):CallSet

Sets up the necessary things for the timer functionality to operate. If the timer has already been setup before, it will be "set-down".

Parameters
